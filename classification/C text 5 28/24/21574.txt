2.5.3变量定义
注意，C99标准中接受多个限定符（即幂等限定符：idempotent type qualifiers)，当同一限定 符在同一声明语句中出现多次（无论是直接出现还是通过一个或多个typedrf)时，其意义与该限 定符仅出现一次时相同。
const const double PI=3.14159;	/* c99 允许，c89 不允许 */
一个变量有一个名字，其本质代表计算机内存的某一存储单元，可以把值放入其中，并访问它。 程序中用到的所有变量都必须先声明后使用。声明有两个目的：告诉编译器为变量分配适当长度的 存储单元，以及确定变量所允许的运算。声明中首先要指定其类型，然后包含一个或多个该类型的 变量组成的变量表，最后以分号结尾，一般形式为：
类型名变量表；
