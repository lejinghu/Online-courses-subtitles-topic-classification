(6)无符号长整型（unsigned long int 或 unsigned long)。
数据类型符为unsigned long int，也可写成unsigned long，占用4个字节（32位）的内 存单元。最高位不是符号位，而是数据位。unsigned long类型与long类型的区别可参考 unsigned int与int类型区别。
unsigned long int x=4389828； unsigned long y=0XA5CD99AB；
