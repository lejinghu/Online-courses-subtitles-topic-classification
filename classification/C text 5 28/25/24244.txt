其中，g被定义为浮点型常量，const是常量的说明符。常量的定义只需要在变量的定 义中增加一个修饰符号const即可。需要注意，在定义常量的同时，需要为常量赋值；不可 以在定义了常量之后，为常量赋值，否则会出现错误。
下面我们通过一个例子来说明为什么要使用常量。例如，如果任意给定的一个物体的 质量为m，求该物体的重量G。重量与质量之间存在的关系是：G=mg。其中，g是一个不 变的量，一般等于9.8。在物理运算中，可能多次使用g。为了方便，可以将9.8定义为常 量。在C语言中，我们通常将常量用以下的形式来定义：
const float g = 9.8;
2.	避免输入错误
下面我们再举一个例子来说明使用常量的好处避免输入出错。例如，在计算一个 圆的面积时，可以将圆周率定义为常量，代码如下： const float PI = 3.1415926;
