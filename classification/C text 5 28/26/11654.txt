在第5章己经看到，如何利用下面的表达式来判断字符变量C是否包含小 写字母：
在关系表达式和算术表达式中，经常使用字符变量和常量。为在这些情况 下正确使用字符，需要理解C编译器是如何处理它们的。
在C语言中，只要在一个表达式中使用字符常量或变量，就会自动将其转 换为一个整数值，之后也将一直作为整数值进行处理。
c >= 'af && c <= 'z'
前面曾经提到，这种表达式可以用在一种采用ASCI丨字符表示的系统上，因为 小写字母在ASCII中是顺序表示的，两个字母之间不存在其他字符。上述表达 式的第一部分将c的值与字符变量'a的值进行比较，它实际上是将c的值与 字符'a'的内部表示进行比较。在ASCII中，字符的值为97,字符'b?的 值为98,以此类推。因此，对于c中包含的任意小写字母，表达式c>=，ai 都是TRUE (不为0),这是因为它的值大于或等于97。但是，因为有些非小写 字母的ASCI丨值也大于97 (比如左右小括号），所以这一检测必须在另一端设 立界限，以确保该表达式的结果仅对于小写字母为TRUE。为此，还要将c与 字母'z'进行比较，其ASCII值为122。
