由此可见，字符数据在内存中的存放形 式与整数的存放形式是类似的，因此可以把 字符型数据看成整型数据，即它们之间可以 通用。这样，C语言允许对字符变量赋以整 型值，也允许对整型变量赋以一个字符。而 字符数据在输出时既可以选择以字符形式来
char a,b; a=,x'; b='y';
执行上面语句后，变量x和y的内存空间如图4.7所示。
X：	0	1	1	1	1	0	0	0
