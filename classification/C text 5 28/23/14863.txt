C语言是由Dennis Ritchie创造并首先在配备UNIX操作系统的DEC PDP- 11计算机上实现的。C语言是 早期计算机语言BCPL(目前在欧洲还有应用）发展过程的产物。BCPL由MartinRichanls开发，影响了 Ken Thompson首创的B语言。B语言导致了 C语言在70年代的发展。
本章描述程序设计语言C的概貌、起源、应用以及构成原则，主要适于初学者。
1.1	C语言的简要历史
多年来,UNKV系统配备的C语言一直是C的公认标准，在Brian Kemighan和Dennis Ritchige合著的《C语 言程序设计》〈TTie C Programming Language,Prentice^hall，1987) —书中有描述。随着微型计算机的普及，出现了 大批C语言系统。令人吃惊的是，其中多数系统接受的源程序都能高度兼容（B卩，为其中一台计算机编写的 程序可以成功地用另一台去编译）。然而，没有统一标准，必然存在差异。为了克服这种不利局面，ANSI于 1983年初夏成立了一个委员会，由该委员会制订一劳永逸的定义C语言的标准。标准化过程花了 6年时间 (比任何人想象的时间都要长）。
ANSI C标准于丨989年被完全采用，并于1990年推出第一■个范本。ISO (International Standards Organization, 国际标准化组织)也采用此标准，该标准一般称为ANSI/ISO Standard C。1995年，C标准的第一次修订版被采 纳，其中增加了一些新的库函数。在此基础上，1989年的C标准成为标准C++的基础文档，并定义C为C+ + 的子集。1989年定义的C标准一般称为C89〇
