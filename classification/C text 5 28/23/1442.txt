最初，C语言没有统一的官方标准可供遵循。1978年，Brian Kemighan和Dennis Ritchie完成 了影响深远的《The C Programming Language》第一版的写作，该书的附录中提供了 C语言参考手 册。在以后的几年里，该书一直被广泛作为C语言的实现规范，成为非官方的C语言标准。这本 书提出的C语言标准通常被简称为“K&RC”，其中，“K”和“R”是两个作者名字的首字母。
C语言发展到现在已经经历了三个标准，分别为K&RC、C89或ANSIC和C99。

2.	C89 或 ANSIC
C语言在随后短短几年的应用中得到了很快的发展。实际上使用的C语言早已超越了 K&R C描述的范围，同时，K&RC在语言的许多细节上也没有达到足够精确的要求，这使标准化C 语言成为迫切的要求。于是，在1983年夏天，美国标准化组织（ANSI)成立了一个C语言工 作小组，开始了 C语言的标准化工作。小组主要负责确认C语言的常用特性，但也对语言本 身做了一些修改，并引入了一些新的特性。
