如果使用布尔型变量，通常的习惯是用变量自身的值作为判断条件。也就是说用：
set inword to false	// 将标志置 0 (或假）
这种方法在每个单词开始时将inword设为1 (真），而在每个单词结束日时将其设为 0 (假）。仅在该标志从0变为1时寸单词计数。如果在你的系统上可以使用_Bool 型变量，可以包含stdbool.h头文件并用boo丨作为inword的类型，取值分别为true 和false。否则，就使用int类型，取值为0和1.
